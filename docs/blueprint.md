# **App Name**: MobileFlow

## Core Features:

- Personalized Welcome: Display a welcome message to the user using their name.
- Customer Management Card: Provide a card-based menu for main navigation: 'Gestion de Clientes' (Customer Management).
- Search Card: Provide a card-based menu for main navigation: 'Buscador' (Search).
- Inventory Card: Provide a card-based menu for main navigation: 'Inventario' (Inventory).
- Sync Card: Provide a card-based menu for main navigation: 'Sincronizar' (Synchronize).

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence.
- Background color: Very light blue (#E8EAF6), almost white, to provide a clean backdrop.
- Accent color: Muted purple (#9575CD) for a modern touch.
- Card-based layout for the main menu items, ensuring touch targets are clearly defined on the mobile screen.
- Use a clean, sans-serif font for readability on mobile devices.
- Simple and clear icons to visually represent each menu option on the cards.
